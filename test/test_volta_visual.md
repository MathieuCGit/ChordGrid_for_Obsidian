# Test visuel des voltas

## Exemple simple avec volta 1-3

```chordgrid
4/4 ||: C[4 88_4 4] | % |.1-3 G[%] | % :||.4 G[4 4 4 4] ||
```

Le volta "1-3" devrait :
- Commencer sur la barre de mesure entre le deuxième C (%) et le premier G
- Se terminer sur la barre de mesure après le deuxième %
- Avoir un crochet fermé à droite (car avant :||)

Le volta "4" devrait :
- Commencer sur la barre de mesure après :||
- Se terminer sur la barre de mesure avant ||
- Avoir un crochet ouvert à droite (car après :||)

## Exemple simple

```chordgrid
4/4 ||: C[4 4 4 4] |.1 G[4 4 4 4] :||
```

Le volta "1" devrait commencer sur la barre entre C et G.
